# GIT

1. 저장소(repasitory)만들기 - git이 관리하는 폴더(directory)

   이미 init 한 repo에서 또 init 하지 않는다!! 

   ``` shell
   $ git init 
   ```

2.  파일을 스테이징하기

   ```shell
   $ git add 파일명
   ```

3.  파일을 커밋하기 

```shell
$ git commit -m '커밋 메세지 내용(변경사항을 요약해서 적음)'
```

4. 상태 확인하기

```shell
$ git status
```

5. 내가 누군지 정보 입력하기

```shell
$ git config --global user.name 유저이름
$ git config --global user.email 유저이메일
```
